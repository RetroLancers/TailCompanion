# TailCompanion

A specialized repository for Hytale modding, featuring a comprehensive set of agent skills for designing, implementing, and managing Hytale entities, pets, and systems.

## Hytale Modding Skills

This repository implements 16 specialized skills for Hytale development:

### Core Engineering
- **hytale-entity-design**: ECS architecture and Codecs.
- **pet-ai-state-machine**: AI behavior implementation.
- **pet-stats-and-decay**: Server-side stat management.
- **pet-persistence**: Data serialization and migrations.

### Interaction & Gameplay
- **command-driven-interactions**: Custom slash commands.
- **pet-feedback-and-emotion**: Visual feedback and empathy.

### Asset & Cosmetic
- **pet-asset-pipeline**: Art pipeline conventions.
- **cosmetic-attachment-system**: Bone and socket attachments.
- **cosmetic-monetization-safety**: EULA compliance.

### Automation & AI
- **blender-python-generation**: Procedural asset creation.
- **pet-dsl-specification**: Structured asset generation.
- **asset-validation-and-qc**: Quality control automation.

### Product & Scaling
- **pet-framework-architecture**: Extensible system design.
- **content-cadence-planning**: Release management.
- **marketplace-packaging**: Distribution preparation.
- **anti-scope-creep-guardrails**: Scope management.
